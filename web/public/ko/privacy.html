개인정보처리방침

최종 수정일: 2026년 3월 22일

Palette Box(이하 "회사")는 CodeSafer 브랜드로 codesafer.org 웹사이트와 cleaner-code MCP 서버 소프트웨어를 운영합니다. 본 개인정보처리방침은 회사가 이용자의 정보를 어떻게 수집, 이용, 보호하는지 설명합니다.

1. 수집하는 정보

계정 정보: 유료 플랜을 구독할 때, 결제 대행사 Paddle이 이름, 이메일 주소, 결제 정보를 수집합니다. Paddle은 모든 거래의 판매 기록 대행자(Merchant of Record)로서 활동합니다.

이용 데이터: 유료 API 사용자의 경우, 요금제 한도 적용을 위해 청구 기간별 API 호출 횟수를 추적합니다. 분석 요청 처리 시간을 초과하여 분석 대상 코드의 내용을 서버에 저장하지 않습니다.

로컬 분석: 정적 분석은 사용자의 로컬 머신에서 완전히 실행됩니다. 정적 분석 중에는 어떠한 코드도 서버로 전송되지 않습니다.

AI 분석 (API): 클라우드 기반 AI 분석 API 사용 시, 코드 스니펫이 분석을 위해 서버로 전송됩니다. 이 스니펫은 메모리에서 처리되며 영구 저장되지 않습니다.

2. 정보 이용 목적

  • 서비스 제공 및 유지
  • 구독 및 API 접근 관리
  • 요금제에 따른 사용량 제한 적용
  • 서비스 업데이트 안내 및 지원 요청 응대
  • 탐지 모델 및 서비스 품질 개선

3. 결제 처리

모든 결제는 판매 기록 대행자인 Paddle이 처리합니다. 회사는 신용카드 정보를 직접 수집하거나 저장하지 않습니다. 결제 데이터 처리에 대한 자세한 내용은 Paddle의 개인정보처리방침을 참고하시기 바랍니다.

4. 정보 공유

회사는 이용자의 개인정보를 제3자에게 판매, 거래 또는 임대하지 않습니다. 다음의 경우에만 정보를 공유할 수 있습니다:

  • 결제 처리를 위해 Paddle과 공유
  • 법률 또는 법적 절차에 의해 요구되는 경우
  • 회사의 권리, 개인정보, 안전 또는 재산을 보호하기 위한 경우

5. 데이터 보안

회사는 암호화된 연결(TLS/SSL), 안전한 API 키 인증, 접근 제어 등 업계 표준 보안 조치를 시행합니다. 그러나 인터넷을 통한 전송 방법이 100% 안전한 것은 아닙니다.

6. 데이터 보관

계정 데이터는 계정이 활성 상태인 동안 보관됩니다. API 사용 로그는 90일간 보관됩니다. AI 분석을 위해 제출된 코드는 처리 시간 이후 저장되지 않습니다. 언제든지 데이터 삭제를 요청할 수 있습니다.

7. 이용자 권리

이용자는 다음과 같은 권리를 가집니다:

  • 회사가 보유한 개인 데이터에 대한 접근권
  • 부정확한 데이터의 정정 요청권
  • 데이터 삭제 요청권
  • 데이터 처리에 대한 이의 제기 또는 제한 요청권
  • 데이터 이동권

이러한 권리를 행사하려면 paletteboxofficial@gmail.com로 연락해 주세요.

8. 쿠키

본 웹사이트는 필수 기능에 한해 최소한의 쿠키만 사용합니다. 개인정보를 수집하는 추적 쿠키나 제3자 분석 도구를 사용하지 않습니다.

9. 아동의 개인정보

본 서비스는 16세 미만의 개인을 대상으로 하지 않습니다. 아동의 개인정보를 고의로 수집하지 않습니다.

10. 방침 변경

본 개인정보처리방침은 수시로 업데이트될 수 있습니다. 변경 사항이 있을 경우 이 페이지에 새로운 방침을 게시하고 "최종 수정일"을 업데이트합니다.
