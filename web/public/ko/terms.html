이용약관

최종 수정일: 2026년 3월 22일

본 이용약관("약관")은 Palette Box(이하 "회사")가 CodeSafer 브랜드로 제공하는 제품 및 서비스의 이용에 관한 사항을 규정합니다. codesafer.org 웹사이트, cleaner-code MCP 서버, AI 분석 API(이하 통칭 "서비스")를 이용함으로써 본 약관에 동의하게 됩니다.

1. 서비스 설명

CodeSafer는 소스 코드에 숨겨진 악성 패턴을 탐지하는 코드 보안 분석 도구를 제공합니다. 서비스에는 다음이 포함됩니다:

  • 정적 분석: 유니코드 공격, 난독화, 의존성 위험 등을 위한 로컬 패턴 기반 스캐닝
  • AI 분석: CodeBERT를 활용한 클라우드 기반 딥러닝 위협 탐지
  • MCP 서버: Model Context Protocol을 통한 AI 코딩 도구와의 연동

2. 계정 및 API 키

유료 플랜은 Paddle을 통해 관리되는 유효한 구독이 필요합니다. 구독 시 프리미엄 기능에 접근하기 위한 API 키가 발급됩니다. API 키의 기밀 유지는 이용자의 책임입니다. 권한 없는 제3자와 API 키를 공유하지 마세요.

3. 구독 플랜

다음과 같은 플랜을 제공합니다:

  • Community (무료): 무제한 정적 분석, 세션당 AI 분석 10회
  • Dev ($9/월): 월 200회 AI 분석, API 접근, 이메일 지원
  • Team ($29/사용자/월): 월 2,000회 AI 분석, CI/CD 통합, 우선 지원
  • Enterprise: 맞춤 가격, 온프레미스 배포, 무제한 분석

가격은 USD 기준이며 Paddle이 결정하는 해당 세금이 적용될 수 있습니다.

4. 결제 조건

모든 결제는 판매 기록 대행자인 Paddle이 처리합니다. 구독 시 Paddle의 이용약관에도 동의하게 됩니다. 구독은 선택한 결제 주기에 따라 월간 또는 연간으로 청구됩니다. Paddle의 고객 포털을 통해 언제든지 구독을 취소할 수 있습니다.

5. 환불 정책

모든 유료 플랜에 30일 환불 보증을 제공합니다. 서비스에 만족하지 않는 경우, 최초 구매 또는 갱신 후 30일 이내에 전액 환불을 요청할 수 있습니다. 환불은 Paddle을 통해 7영업일 이내에 처리됩니다.

환불을 요청하려면 구독 정보와 함께 paletteboxofficial@gmail.com로 연락해 주세요.

30일 기간 이후의 환불은 개별 사안에 따라 회사의 재량으로 제공됩니다.

6. 이용 제한

이용자는 다음 행위를 하지 않을 것에 동의합니다:

  • 불법적인 목적으로 서비스를 사용하는 행위
  • AI 모델을 리버스 엔지니어링, 디컴파일 또는 디어셈블하려는 행위
  • API 키를 공유하거나 재배포하는 행위
  • 제한을 우회하기 위한 자동화 수단으로 플랜 사용량 한도를 초과하는 행위
  • 경쟁 제품 개발에 서비스를 이용하는 행위
  • 서비스 인프라를 방해하거나 중단시키는 행위

7. 지식재산권

소프트웨어, AI 모델, 탐지 알고리즘, 문서를 포함한 서비스는 Palette Box가 소유하며 지식재산권법에 의해 보호됩니다. 구독은 플랜에 따라 서비스를 사용할 수 있는 제한적이고, 비독점적이며, 양도 불가능한 라이선스를 부여합니다.

8. 보증 면책

서비스는 명시적이든 묵시적이든 어떠한 종류의 보증 없이 "있는 그대로" 및 "이용 가능한 그대로" 제공됩니다. 분석이 모든 보안 위협을 탐지하거나 서비스가 중단 없이 또는 오류 없이 제공될 것을 보장하지 않습니다. CodeSafer는 보조적인 보안 도구이며 코드 보안 검토의 유일한 수단이 되어서는 안 됩니다.

9. 책임 제한

법률이 허용하는 최대 범위 내에서, Palette Box는 서비스 사용으로 인해 발생하는 간접적, 부수적, 특별, 결과적 또는 징벌적 손해(이익, 데이터 또는 사업 기회 손실 포함)에 대해 책임을 지지 않습니다. 회사의 총 책임은 청구일 이전 12개월 동안 서비스에 대해 지불한 금액을 초과하지 않습니다.

10. 약관 변경

회사는 언제든지 본 약관을 변경할 권리를 보유합니다. 중요한 변경 사항은 웹사이트에 업데이트된 약관을 게시하여 안내합니다. 변경 후 서비스를 계속 사용하면 새로운 약관에 동의한 것으로 간주됩니다.

11. 해지

이용자가 본 약관을 위반할 경우 서비스 접근을 중단하거나 해지할 수 있습니다. 이용자는 언제든지 구독을 취소할 수 있습니다. 해지 시 API 키가 비활성화되고 프리미엄 기능에 대한 접근이 중단됩니다.

12. 준거법
